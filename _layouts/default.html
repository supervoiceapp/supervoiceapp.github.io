SUPERVOICE: Text-Independent Speaker Verification Using Ultrasound Energy in Human Speech

SUPERVOICE is a text-Independent speaker verification system which ultilizes ultrasound in human speech. It can be adapted to any Deep Neural Network based speaker identification model. With integrating the ultrasound of genuine, SUPERVOICE is capable to differentiate between genuine human and machine based replay attacks, or other adversarial attacks that induced by machine. This website demonstrates the dataset that we used to evaluate SUPERVOICE.

    Dataset Size

    78 person, 7800 audio samples, 100 audios/person

    Dataset Design

    There are 4 types of sentences included by the dataset.

  • ★Common:The common type contains 5 sentences, two of which are from the dialect set of TIMIT dataset [1], while the other three sentences are “This is a test", “Please identify me", and “OK, please recognize my voice". For this type of sentence, each participant will speak every sentence twice, resulting in 10 recordings.
  • ★Compact:The compact type has 200 sentences in total. They are randomly chosen from the TIMIT dataset, and designed to provide a good coverage of phonemes, with extra occurrences of phonetic contexts that are considered either difficult or of particular interest. Each participant reads 40 sentences from this type.
  • ★Fricative:The fricative commands are col- lected from the website ok-google.io, which provides com- monly used commands on Google Assistant. We select 50 sentences that have fricative phonemes, e.g., “find my phone", “turn on my lights". Each speaker randomly picks 25 sentences from this type of commands.
  • ★Non-fricative:Similar to the fricative com- mands, the non-fricative commands are also collected from ok-google.io, which do not contain any fricative phonemes. We randomly pick 25 sentences and ask every participant read 25 sentences from this type
